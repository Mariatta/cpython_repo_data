Contribution data for the CPython repository

Data collected by Mariatta, using Python 3.9 and gidgethub library to fetch data using GitHub API. Charts are generated using Google Charts.
Source code

Other interesting dates

  • CPython moved to GitHub in Feb 2017.
  • PyCon US: in May of 2017, 2018, 2019, 2020
  • Python core dev sprint: In Sept 2017, 2018, 2019, and virtually in Oct 2020
  • October of each year is also Hacktoberfest.
  • Guido retired as BDFL: July 2018
  • Feb 2019: The first steering council members were elected

Notes

  • January 2021 data is not complete.
  • Seems like there's usually a peak during times of PyCon US and Python core sprints.
  • Seems like folks normally take time off during summer months :)
  • ... and end of the year holidays.
  • PEP 572 was created on Feb 2018.
  • Somehow.. PR contributions were at all time low in Feb - April 2018...
